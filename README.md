# Parser Unicorn

## Introduction

A facebook fan page parser and word cloud in Python.

## Installation

``` bash
git clone git@github.com:m80126colin/parserUnicorn.git
cd parserUnicorn/
pip3 install requests bottle jieba
```

## Build

Use Node.JS to build website if there is no `dist/` directory.

``` bash
npm install
npm run build
```

## Execution

``` bash
python3 app.py
```
